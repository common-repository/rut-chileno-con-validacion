Agrega el campo Rut en Finalizar Compra de WooCommerce

== Description ==

No requiere mucho detalle: este plugin agrega un campo Rut en el Finalizar Compra de WooCommerce.

Entre sus características (algunas configurables) están:

*   Permite agregar el texto "para DTE" en el campo.
*   Valida la sintaxis del Rut (módulo 11)
*   Permite hacer el campo requerido u opcional
*   Deja el campo en la órden dentro de WooCommerce
*   Guardar el Rut en el perfil del usuario
*   Envía el Rut junto a los correos electrónicos del sitio

== Installation ==

Instalar este plugin es simple:

1. Sube `wc-chilean-bundle.php` a la carpeta `/wp-content/plugins/`
2. Actívale a través del menú 'Plugins' en WordPress
